# SolidInvest Авто-тесты

Проект для автоматического тестирования главной страницы сайта https://solidinvest.ru/.

## Технологии
- Java 11
- Maven
- Selenium WebDriver
- TestNG
- Page Object Model
- Allure Reports
- WebDriverManager (автоустановка драйверов)

## Тест-кейсы
1. **TC-01** - Проверка загрузки главной страницы
2. **TC-03** - Проверка перехода по ссылке "Контакты"

## Запуск тестов локально
```bash
mvn clean test
