# SolidInvest Авто-тесты

Проект для автоматического тестирования главной страницы сайта https://solidinvest.ru/.

## Технологии
- Java 11
- Maven
- Selenium WebDriver
- TestNG
- Page Object Model
- Allure Reports
- WebDriverManager (автоустановка драйверов)

## Запуск тестов локально
```bash
mvn clean test
